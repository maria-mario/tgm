Step 01 ¬∑ Foundation

Google Tag Manager

Install once. Manage every other tag from one dashboard.

‚ÑπÔ∏è
Do this first. GTM is the container for all other tags. You add one snippet to your site, then deploy GA4, Google Ads, Meta Pixel and more ‚Äî without touching your code again.
1
Create your GTM account and container
+

Go to tagmanager.google.com and sign in with your Google account.

  • Click Create Account ‚Äî enter your company name and country
  • Enter your website URL as the Container Name
  • Select Web as the target platform ‚Üí click Create
  • Accept the Terms of Service
‚úì
Your Container ID will appear ‚Äî it looks like GTM-XXXXXXX. You'll need this in the next step.
2
Add the GTM snippets to your website
+

You need two snippets on every page. Replace GTM-XXXXXXX with your Container ID in both.

Snippet 1 ‚Äî inside <head>, as high as possible:

<script>(function(w,d,s,l,i){w[l]=w[l]||[];w[l].push({'gtm.start':

j=d.createElement(s),dl=l!='dataLayer'?'&l='+l:'';j.async=true;j.src=

Snippet 2 ‚Äî immediately after opening <body>:

<noscript><iframe src="https://www.googletagmanager.com/ns.html?id=GTM-XXXXXXX"

Using a website builder?

  • Webflow: Project Settings ‚Üí Custom Code
  • Wix: Settings ‚Üí Custom Code ‚Üí Add to <head>
3
Verify GTM is working
+
  • In GTM click Preview (top right) ‚Üí enter your URL ‚Üí Connect
  • Your site opens with a debug panel ‚Äî confirm GTM is detected

Step 02 ¬∑ Analytics

Google Analytics 4

Track pageviews, events, conversions, and campaign performance via GTM.

1
Create a GA4 property and get your Measurement ID
+
  • analytics.google.com ‚Üí Admin ‚Üí Create Property
  • Enter name, time zone, currency ‚Üí Next ‚Üí choose platform: Web
  • Enter your URL and stream name ‚Üí Create stream
‚úì
Copy your Measurement ID ‚Äî it looks like G-XXXXXXXXXX.
2
Create the GA4 tag in GTM
+
  • GTM ‚Üí Tags ‚Üí New ‚Üí Google Tag
  • Paste your Measurement ID (G-XXXXXXXXXX)
  • Trigger: All Pages ‚Üí Name: GA4 ‚Äì Configuration ‚Üí Save
  • Click Submit ‚Üí Publish
‚ö†Ô∏è
Always publish after saving. Tags only go live when you publish the GTM container.
3
Enable Enhanced Measurement
+

Automatically tracks scrolls, outbound clicks, downloads, video plays ‚Äî no extra code needed.

  • GA4 ‚Üí Admin ‚Üí Data Streams ‚Üí your stream
  • Toggle Enhanced Measurement ON ‚Üí click gear to customise
4
Set up lead magnet conversion tracking
+
  • GTM ‚Üí Triggers ‚Üí New ‚Üí Page View
  • Condition: Page URL contains /thank-you
  • New GA4 Event tag ‚Üí event name: generate_lead ‚Üí attach trigger ‚Üí Publish
  • GA4 ‚Üí Admin ‚Üí Events ‚Üí toggle generate_lead as a conversion

Step 07 ¬∑ Email Campaigns

EmailOctopus

Connect email campaigns to GA4, Google Ads, and Meta using UTM parameters.

‚ÑπÔ∏è
How EmailOctopus connects to your tracking stack. EmailOctopus tracks opens and clicks inside its own dashboard automatically. To see email traffic in GA4, Google Ads, and Meta ‚Äî you add UTM parameters to every link in your emails. When a subscriber clicks through, GA4 reads the UTMs and attributes the visit and any conversions correctly.
1
What EmailOctopus tracks natively (no setup needed)
+

Inside your EmailOctopus dashboard you get these automatically:

  • Open rate ‚Äî how many subscribers opened each email
  • Click rate ‚Äî how many clicked any link
  • Unsubscribes and bounces ‚Äî list health metrics

What EmailOctopus does not track: what happens after the click ‚Äî purchases, form fills, lead magnet downloads, time on site. That's what UTMs + GA4 handle.

2
Set your UTM naming convention
+

A UTM-tagged email link looks like this:

URL ¬∑ UTM Example
https://yoursite.com/lead-magnet?utm_source=emailoctopus&utm_medium=email&utm_campaign=welcome-sequence&utm_content=cta-button

Use this as your reference for every email link:

ParameterValue to useNotes
utm_sourceemailoctopusAlways this ‚Äî identifies the platform
utm_mediumemailAlways "email" for all email traffic
utm_campaignwelcome-sequence
april-newsletter
lead-magnet-launch		The specific campaign or sequence name
utm_contentcta-button
text-link
header-banner		Which link in the email ‚Äî useful if you have multiple CTAs
‚ö†Ô∏è
Be consistent. If you use emailoctopus once and email-octopus another time, GA4 treats them as two separate sources. Pick one format and never change it.
üí°
Build UTM links easily. Use Google's free Campaign URL Builder at ga-dev-tools.google.com/campaign-url-builder to generate clean, properly encoded URLs every time.
3
Add UTM links inside EmailOctopus campaigns
+

EmailOctopus does not auto-append UTMs, so you add them manually when building each email.

  • For each CTA button or text link, paste your UTM-tagged URL instead of the bare URL
  • Use a different utm_content value for each link in the same email
  • Use a different utm_campaign for each email in a sequence ‚Äî e.g. welcome-email-1, welcome-email-2
  • Do the same for your lead magnet delivery emails, re-engagement campaigns, and newsletters
4
View email traffic and conversions in GA4
+
  • GA4 ‚Üí Reports ‚Üí Acquisition ‚Üí Traffic acquisition
  • Filter by Session medium = email to see all email traffic
  • Filter by Session source = emailoctopus to isolate your campaigns
  • The Conversions column shows how many lead magnet downloads each campaign drove
‚úì
Because you set up the generate_lead conversion earlier, GA4 will show you exactly which EmailOctopus campaigns are driving the most leads.
5
Optional: GTM trigger for email-only visitors
+

Fire specific tags or show personalised content only when a visitor arrives from an email link.

  • GTM ‚Üí Triggers ‚Üí New ‚Üí Page View
  • Condition: Query String contains utm_source=emailoctopus
  • Attach this trigger to any tag you only want to fire for email subscribers
6
EmailOctopus limitations to know
+
  • Email opens will never appear in GA4 ‚Äî only clicks that land on your website
  • No native GA4 or GTM integration ‚Äî UTMs are the only bridge between the two platforms
  • No server-side event sending (unlike Klaviyo or ActiveCampaign)
  • If a subscriber has ad/cookie blocking, the UTM visit may not be attributed in GA4
üí°
Cross-reference your data. Compare EmailOctopus click counts against GA4 email sessions. The gap is usually ad blockers or cookie consent declines ‚Äî completely normal.
